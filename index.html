Software Design and Engineering

Project Overview

A mobile application with comprehensive performance testing capabilities and thread-safe implementations.

Original Artifact

A mobile application with three interconnected services (ContactService, TaskService, and AppointmentService) originally developed for CS-320: Software Testing, Automation, and Quality Assurance. The application implements core CRUD operations for managing contacts, tasks, and appointments, with each service maintaining strict validation rules for data integrity.

Enhanced Version

The enhanced version adds comprehensive performance testing capabilities and improves thread safety. Key improvements include replacing ArrayList with ConcurrentHashMap, implementing proper synchronization using ReadWriteLock, and adding sophisticated load testing and metrics collection.

Algorithms and Data Structure

Project Overview

A course advisory system implementing advanced graph algorithms for prerequisite validation and course sequencing.

Original Artifact

A course advisory system originally created in CS-300 Data Structures and Algorithms. The system manages course data and prerequisites using a Binary Search Tree data structure, providing functionality for course queries and prerequisite validation.

Enhanced Version

The enhanced version implements Depth-First Search with Topological Sort to improve the course prerequisite system. This includes cycle detection in prerequisite relationships, proper handling of complex prerequisite chains, and comprehensive test coverage for edge cases.
